git 명령어 사용방법
git init 				: 해당 디렉토리를 Git repository로 초기화.
git config --global user.name " " 	: 해당 버전을 만든 사람 이름.
git config --global user.email " "	: 해당 버전을 만든 사람 이메일.
git add 파일이름.확장자		: 해당 파일을 add.
git add .				: 변경된 파일 전체 add.
git commit -m " 내용 "		: 커밋 하면서 버전 내용설명 적기.
